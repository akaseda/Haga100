送迎管理システム

学生の車・実行委員の車を分けて登録し、茨大・友部駅・荒川沖駅など集合場所ごとの配車表を作成します。

1. Excel / CSVを読み込む

Microsoft Formsの回答Excelをそのまま読み込めます。正式メンバー決定後は、このブラウザに保存しておけば毎回アップロード不要です。

保存先はこのブラウザ内です。別のPCや別ブラウザでは再度Excelを読み込んでください。

2. 列の確認

通常は変更不要です。読み取りが違っていた場合だけ、ここで手動で選び直してください。

3. 集合場所・時間・責任者

実行委員と共有する文面の「☆茨大11:30」「責任者：〇〇」に反映されます。

4. 車の設定

車の定員合計
8人
「実行委員の車」は運転者を学生から選ばなくても使えます。「学生の車」にした場合、運転者にした学生は確定で車組になります。

5. 参加者・学生運転者・イレギュラー対応

学生が運転する場合だけ「学生運転者」にチェックしてください。実行委員の車に乗るだけならチェック不要です。
